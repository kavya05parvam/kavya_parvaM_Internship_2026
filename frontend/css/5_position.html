position property:

it used to control the position of html

        1.position: static - this is default position(cannot use left, right, top, bottom)
        2.position: relative- this follows the original position absolute
        3.position: absolute- this positions the element from the nearest ancestor
        4.position: fixed- it will the html element
        5.position. sticky- it will stick the element when we even scroll the page
